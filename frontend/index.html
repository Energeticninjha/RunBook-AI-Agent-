🤖 RUNBOOK AGENT

Autonomous Incident Resolution Engine powered by local Llama 3

RUNBOOK AGENT is an autonomous agent that monitors infrastructure servers, identifies real-time failures, and walks through recovery runbooks. It uses the Model Context Protocol (MCP) to safely execute diagnostic and repair commands.

🚀 Features

  • Deterministic Autonomy: Safely executes markdown-based operations step by step.
  • AI Classification: Local Llama 3 identifies RISK level (SAFE vs RISKY) for all runbook commands.
  • Human-in-the-Loop: Securely pauses for approvals before executing risky commands.
  • Real-time Dashboard: Continuous terminal output log stream and server metrics tracker.

About

Autonomous DevOps IT Operations Agent bridging AI reasoning with system execution tools via Model Context Protocol (MCP).
